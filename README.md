Django Expense Tracker
=======================
---------
About
-----

A responsive web application for expense tracking and analytics powered by Django
Features

Installed apps:
-----
* django 2.0.2
* django.contrib.admin
* django.contrib.auth
* django.contrib.contenttypes
* django.contrib.sessions
* django.contrib.messages
* django.contrib.staticfiles
* tracker.apps.TrackerConfig
* django_tables2
* django_filters
* bootstrap3
* crispy_forms

Configured URLs:

* ``/add``
* ``/analytics``
* ``/admin/``

Templates:

* ``tracker/expense_form.html``
* ``tracker/from_template.html``
* ``tracker/header.html``
* ``tracker/index.html``
* ``analytics/index.html``

Features:

* Responsive 
* Expense Overview with filtering
* Expense addition
* Expense Analytics by year, month or day
* Bootstrap 3 based
